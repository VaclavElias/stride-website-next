Community Support

If you have a problem learning Stride or need help, please first use the various community channels. Stride developers are busy working in making Stride better, so they can't always answer questions. They will, however, be glad to assist you if you can't get an answer from conventional means.

Commercial support

At the moment we don't provide commercial support.

General inquiries

For other inquiries please use our community channels.

FAQ

A list of Frequently Asked Questions covering various topics.

Report a bug

Sometimes bugs happen, but don’t worry! Developers got your back, just report a bug and we’ll all enjoy a more stable Stride.

If you’d like to help fixing bugs, we have plenty of ways how you can get involved with making Stride better.

Documentation

Stride's user manual covers all aspects of the engine, from installation and getting started to advanced features and best practices, ensuring a smooth learning curve for users.

Tutorials

Access a variety of tutorials and example projects that demonstrate how to use Stride's features effectively, catering to users with different skill levels and experience.
